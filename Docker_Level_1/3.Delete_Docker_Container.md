# Solution

1. ssh into the App Server 3
    ```
    ssh banner@172.16.238.12
    ```
2. Verify container
   ```
   docker ps
   ```
3. Stop container
   ```
   docker stop kke-container
   ```
4. Remove container
   ```
   docker rm kke-container
   ```
5. Verify container was deleted
   ```
   docker ps
   ```
