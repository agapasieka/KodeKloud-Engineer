# Solution

1. Create manifest file httpd-deployment.yaml
   ```
   vi httpd-deployment.yaml
   ```
   ```
    apiVersion: apps/v1
    kind: Deployment
    metadata:
      name: httpd
    spec:
      replicas: 1
      selector:
        matchLabels:
          app: httpd
      template:
        metadata:
          labels:
            app: httpd
        spec:
          containers:
            - name: httpd
              image: httpd:latest
   ```
3. Deploy pod
   ```
   kubectl create -f httpd-deployment.yaml
   ```
